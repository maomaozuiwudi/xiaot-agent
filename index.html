🚀 快速开始

# 1. 下载
$ git clone https://github.com/maomaozuiwudi/xiaot-agent.git
$ cd xiaot-agent
# 2. 安装依赖
$ pip install -e .
$ pip install playwright && playwright install chromium
# 3. 配置 API Key
$ cp config.example.yaml config.yaml
# 编辑 config.yaml 填入 DeepSeek / Kimi 等 Key
# 4. 启动（CLI 模式）
$ python main.py
🐱 小t Agent 启动成功！
# 或启动 Web 界面（浏览器访问 http://localhost:7860）
$ python main.py --web
